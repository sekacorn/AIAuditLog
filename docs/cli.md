# CLI

Commands:

- `validate EVENT`: validate one event JSON file.
- `inspect EVENT`: print a compact JSON inspection.
- `append LOG --event EVENT`: append with chain integrity.
- `append LOG --event EVENT --lock`: append with a simple lock file.
- `verify LOG`: verify a chain.
- `query LOG`: filter by event type, status, stream, and limit.
- `summarize LOG`: summarize event types and outcomes.
- `checkpoint create LOG --output PATH`: create a checkpoint.
- `checkpoint verify CHECKPOINT --log LOG`: verify a checkpoint.
- `export LOG --format jsonl|json|csv|markdown --output PATH --compress`: export events, optionally with gzip.
- `redact LOG --profile PROFILE --output PATH`: write redacted JSONL.
- `compress PATH`: create a deterministic gzip copy.
- `rotate LOG`: rotate a stream to the next numbered path.
- `migrate EVENT --output PATH`: migrate an early alpha event document.
- `schema list`: list schemas.
- `schema export --output DIR`: write JSON Schema files.
- `key generate`: create an Ed25519 development key pair.
- `key generate --force`: overwrite existing key files.
- `sign CHECKPOINT`: sign a checkpoint in place.
- `signature verify CHECKPOINT`: verify checkpoint signatures.

Do not commit private keys generated by `key generate`.
