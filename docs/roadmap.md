# Roadmap

## 0.1.0a2 Implemented

- RFC8785-backed canonicalization and an RFC8785 sample vector.
- Signed checkpoint policies.
- Stream rotation.
- Deterministic gzip exports.
- Richer privacy profiles.
- Lock-file guarded append support.
- Schema migration helpers.

## 0.1.0a3 Implemented

- Forge-like dictionary adapter.
- AgentPolicyPack-like decision adapter.
- ModelSwapBench-like benchmark context adapter.
- OpenOntologyLite-like context adapter.
- PrivateAIStack-like metadata adapter.
- OpenTelemetry/W3C Trace Context helpers.

These helpers are neutral adapters and are not claimed as externally tested
against sibling packages.

## 0.2

- Signed stream manifests.
- External checkpoint anchoring.
- Optional PostgreSQL storage.
- Retention policies.
- Audit package bundles.
- Provenance attestations.
- Organization-specific event profiles.
- SIEM exporters.

## 1.0 Considerations

- Stable schema.
- Compatibility policy.
- Formal canonicalization profile.
- Migration policy.
- Interoperability test suite.
- Security review.
